foxcmds






This is foxcmds.

Foxcmds is an custom terminal for linux. I also made versions termux users, but that version is most of the time a coulpe versions behind and dont gets updates that fast. For people that have there user-directory in a different place than '/home/username', i also have an custom starting-script that assigns the root and home-directory different so you can choose them in the config. So, versions for all different types of need!
